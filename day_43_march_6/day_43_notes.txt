CRON
=====
allows us to schedule task in node.js using crontab syntax.


https://www.npmjs.com/package/node-cron


1. npm i node-cron
2.Add the below code
var cron = require('node-cron');
cron.schedule('* * * * *', () => {  // * Every
  console.log('running a task every minute');
});



1. second (optional)
2. minute
3. hour
4. day of month(date)
5. month
6. day of week

0 0 1 1 *        Every Year jan 1st
